# Feature: Add social sharing
- Add an icon next to post/comment icon for a user to share a subxeuron, a publication or an event via
  - unique URL
  - generate a QR code fo sharing
- For each of the subxeuron, publication or event,
  - user can share a specific post, poll, questions via
    - unique URL and its associated QR code
- By default, when a user click on the "share" button or icon, it copies the URL underneath, then "display URL copied"
- when "share" icon is clicked, an QR code is generated on the fly and added to that specific post.
